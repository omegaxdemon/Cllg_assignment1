STORAGE


Storage refers to holding information somewhere. The different types of storage devices are:

1.RAM

The RAM is also known as the Random Acess Memory.
It is a short term or volatile memory and its data
is automatically removed once the computer is turned off.
The RAM is located inside the CPU case on the motherboard.
The motherboard, along with the RAM, also holds the
electronic components, secondary memory and other components.

2.ROM

The ROM is also known as the Read Only Memory.
It is a long term or non-volatile memory and its data
is not removed even when the computer is turned off.
The ROM is located inside the CPU case on the motherboard.
The motherboard, along with the ROM, also holds the
electronic components, secondary memory and other components.
